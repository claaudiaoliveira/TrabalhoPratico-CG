Descrição:
Aplicação para lançamento de foguete. No início é possível controlar o astronauta dentro da plataforma, 
e após uma interação é possível controlar o próprio foguete.

Formas de Interação:
r - reinicia a cena (em ambos os modos)

Modo Astronauta: 
w, a, s, d - controlam o movimento
barra de espaço - altera para o modo foguete

Modo Foguete:
seta pra cima - acelera/lança o foguete verticalmente
seta para baixo - rotaciona o foguete verticalmente no ar (usada apenas após a abertura do paraquedas)
setas para esquerda e direita - rotaciona o foguete horizontalmente no ar (usada apenas após a abertura do paraquedas)
